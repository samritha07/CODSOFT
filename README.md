# Machine Learning Projects

This repository contains three beginner-level Machine Learning projects developed using Python and Google Colab. These projects demonstrate data preprocessing, exploratory data analysis (EDA), model building, evaluation, and prediction using popular machine learning techniques.

## Projects Included

### 1. Titanic Survival Prediction
- Built a classification model to predict whether a passenger survived the Titanic disaster.
- Performed data preprocessing and exploratory data analysis.
- Used passenger information such as age, gender, passenger class, fare, and embarkation details.
- Evaluated the model using accuracy, classification report, and confusion matrix.

### 2. Movie Rating Prediction
- Developed a regression model to predict movie ratings.
- Used features such as genre, director, and actors.
- Performed data cleaning, feature engineering, and exploratory data analysis.
- Evaluated the model using MAE, RMSE, and R² Score.
- Analyzed feature importance to understand factors influencing movie ratings.

### 3. Iris Flower Classification
- Built a classification model to classify Iris flowers into different species:
  - Setosa
  - Versicolor
  - Virginica
- Used sepal and petal measurements as features.
- Performed exploratory data analysis and visualization.
- Evaluated the model using accuracy, classification report, and confusion matrix.
